### ショートカットキー

#### 基本・全体(mac)
@ul[](false)
- ⌘ + space = spotlightを開く(例：スラック)
- fn + backspace = カーソルの<u>右から</u>消す
- tab = 次
- ⌘ + enter = 送信する・提出など
- ctrl + ←,→ = スクリーンの移動
- ⌘ + shift/option + ←,→ = タブの切り替え
- ⌘ + w = タブを閉じる
- ⌘ + q = アプリを閉じる
- ⌘ + shift + 5 = スクリーンを録画
@ulend

### コードスニペット

+++

#### コードスニペットとは？
普段使うモジュールのショートカット

+++

VCSでコードスニペット作って、使ってみよう！

+++
@[1]
@[2]
@[3]
@[4-5]
@[1-5]
```
"c-band が c-row と c-container を囲むスニペット": { // スニペット名
	"prefix": "band-container-row", // 呼び出すコマンド
	"body": ".c-band>.c-container>.c-row", // はき出されるされるコード
	"description": "htmlファイルにc-band が c-row と c-container を囲むを吐き出す" // スニペットの説明
},
```


+++

メリット：毎回同じことを書かなくてもいい<br>
デメリット：スニペットを覚えること

+++
